# Nuo Model Viewer

A simple Wavefront OBJ viewer.

## Latest Update

* Adjustable shadowing PCF and bias to alliviate artifacts.
* Preliminary shadowing for two of the four light sources.

* Normal texture.
* Light source configuration.
* Multiple light sources.
